## Description

[Komik API](https://github.com/thoriqwildan/Komik-API) is a backend system developed using [NestJS](https://nestjs.com/) and [Prisma](https://www.prisma.io/) as an ORM to manage digital comic data. This API is designed to handle various operations related to comic management, such as uploading chapters in ZIP form, storing comic metadata, managing chapter images, and adding bookmark features for users.

## Project setup

before starting, first clone this repository.

```bash
$ git clone https://github.com/thoriqwildan/Komik-API.git
```

Next, install the package and migrate the database.

```bash
$ npm install
$ npx prisma migrate dev
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
